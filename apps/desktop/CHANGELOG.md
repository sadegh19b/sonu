# SONU Desktop Changelog

All notable changes to the SONU Desktop application will be documented in this file.

## [1.0.0] - 2026-01-11

### Initial Version
- Electron + Python + FasterWhisper architecture
- Hold-to-dictate and toggle modes
- Model management (tiny → large-v3)
- System tray integration
- Global hotkey support

### Planned (v2.0.0)
- Migrate to Tauri + Rust + whisper.cpp
- Post-processing features (like Wispr Flow)
- Improved performance and smaller binary
