# SONU Brand Guidelines

## Logo Usage

### Primary Logo
The SONU logo consists of a microphone icon and wordmark. Always use them together for maximum brand recognition.

**Colors:**
- Icon Background: `#27272a` (brand-800)
- Icon Elements: `#a1a1aa` (brand-400)
- Text: `#fafafa` (brand-50)

### Logo Sizes

| Size | Icon | Usage |
|------|------|-------|
| sm | 24x24px | Inline text, compact UIs |
| md | 32x32px | Headers, navigation |
| lg | 48x48px | Hero sections, splash screens |
| xl | 64x64px | Loading screens, about page |

### Clear Space
Maintain a minimum clear space around the logo equal to the icon size on all sides.

### Minimum Size
Never display the logo smaller than 24x24px for the icon portion.

## Typography

### Font Family
- **Primary:** Geist Mono (monospace)
- **Fallback:** ui-monospace, SFMono-Regular, Menlo, Consolas

### Font Sizes
```
text-xs:  0.75rem  (12px) - Captions, labels
text-sm:  0.875rem (14px) - Body small
text-base: 1rem    (16px) - Body
text-lg:  1.125rem (18px) - Headings
text-xl:  1.25rem  (20px) - Large headings
```

### Font Weights
```
font-normal:  400 - Body text
font-medium:  500 - Emphasis
font-semibold: 600 - Headings, buttons
font-bold:    700 - Strong emphasis
```

## Iconography

- **Library:** Lucide React
- **Size:** Match text size (e.g., sm text = 16px icon)
- **Color:** Inherit from parent or use brand colors
- **Stroke Width:** 1.5px default, 2px for emphasis

## Do's and Don'ts

### ✅ Do
- Use the logo on dark backgrounds
- Maintain aspect ratio
- Use approved colors only
- Provide adequate clear space
- Use Geist Mono font

### ❌ Don't
- Change logo colors
- Stretch or distort the logo
- Use on busy backgrounds
- Place text too close to logo
- Use unapproved fonts

## Application Icons

All application icons should follow the same design language:
- Rounded square container
- Microphone motif
- Zinc color palette
- Consistent sizing across platforms

### Platform-Specific Sizes

**macOS:**
- 16x16, 32x32, 64x64, 128x128, 256x256, 512x512, 1024x1024

**Windows:**
- 16x16, 24x24, 32x32, 48x48, 64x64, 128x128, 256x256

**Linux:**
- 16x16, 22x22, 24x24, 32x32, 48x48, 64x64, 128x128, 256x256

**Tray Icons:**
- 16x16, 32x32 (all platforms)
